小游戏说明：
一共8道题，可以用鼠标点击选择，也可以用上下按键选择。
英雄头像所在位置表示你的选择。
可通过点击左箭头和右箭头前进和后退，也可以用左右按键选择。
按ESC退出程序。
要看英文版，请把po文件夹删除（或更名，比如改名为_po）。
如果不能显示中文，请将你电脑系统的国家区域设置到“中国-中文”（zh_CN）。
